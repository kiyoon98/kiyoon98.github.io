Banking/Finance

Examine a hypothetical savings account and implement the "Keep the Change" program.

Keep the Change is a program offered by Bank of Maryland. In this project, you're asked to create a Python program that implements the program on sample bank account data.

Here's how Keep the Change works. Each purchase is rounded up to the nearest dollar and the difference is moved from your checking account to your savings account. For example, if you buy a coffee for $2.49 using your debit card, the purchase is rounded up to the nearest dollar (in this case $3) and the difference ($3 - $2.49 = 51 cents) is moved from your checking account to your savings account. Bank of Maryland says that most people don't actively contribute to their own savings accounts. Keep the Change helps you save money without thinking about it.

In this project, you'll be given a sample checking account statement listing all the purchases made over one year. Your task is to create a Python program that:

  • Reads the data from the provided text file (fdata.txt)
  • Each line of the text file contains the date the purchase was made and the amount. You are asked to split each line apart and store the date and amount in separate variables
  • For each purchase, compute the change that should be moved to the savings account
  • Keep a running total of how much has been moved to the savings account
  • Print to the screen the total amount of money moved to the savings account
  • Create and write to an output file where you store how much was moved on each date as well as the grand total
